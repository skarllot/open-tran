XML RPC

Z serwisu można korzystać dowolnym programem, który potrafi wykonać zdalne wywołanie przez protokół XML RPC. Należy w tym celu skorzystać z metody suggest, która pobiera dwa argumenty: pierwszy to fraza do tłumaczenia, a drugi to dwuliterowy kod języka, na który ma zostać przełożona (dla języka polskiego - "pl").

W Pythonie robi się to na przykład tak: 

      from xmlrpclib import ServerProxy

      server = ServerProxy("http://open-tran.eu")
      for sugg in server.suggest("save as", "pl"):
          print sugg

Wkrótce postaram się umieścić prosty edytorek plików .po, który będzie przy pomocy tego mechanizmu pobierał podpowiedzi tłumaczeń poszczególnych fraz.
