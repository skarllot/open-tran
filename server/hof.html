Software

  1. Translate Toolkit
    Translate Toolkit provides complete support for many file formats. It is used to import several gigabytes of translations to one database.
  2. Debian GNU/Linux
    Hosting and development machines run Debian OS.
  3. Python
    The server, as well as the importing scripts are developed in Python.
  4. SQLite
    More than 50MM words in over 10MM phrases stored in databases of ~3GB. SQLite handles it on a virtual machine with only 128MB of RAM.
  5. Emacs
    The ultimate development environment
