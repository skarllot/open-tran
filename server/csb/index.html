Dolmaczënk całownëch lopków

Brëkùjąc fòrmularza niżi mòżé wësłac na serwerã lopk .po do dolmaczëniô (serwis òpierti je na translation toolkit ë òbsłëdżiwùje wszëtczé fòrmatë wspieróné bez tã bibliotekã).

Lopk:

Dolmaczënk pòjedińczëch frazów

  1. Do mòdernëch przezérników mòże dodac so search engine (sprôwdzë bédowóné dolmaczënczi).
  2. Gwësno to niżódnô krajemnota, że òczénkò szëkbë prosto wprowôdzô adresã w sztaturze http://csb.open-tran.eu/suggest/search engine.
  3. Brëkòwnice Firefoksa mògą dodac załóżkã z kluczowim słowã. Żebë to zrobic, mùszimë wlezc w sprôwianié załóżkama ë dodac nową na nen ôrt:



    Òd terô pòwtôrzné kòżdą razą wpisanié adresë w sztaturze tran fraza sprawi aùtomatné wëwòłanié szëkbë.

XML RPC

Z serwisu mòże brëkòwac równojaką programą, co rozmie zrobic daleczé wëwòłanié przez protokół XML RPC. Nót je w tim célu zwënégòwac metodã suggest, jakô pòbiérô dwa argùmentë: pierszi to fraza dlô dolmaczënkù, a drëdżi to dwa- abò trzëlëterowi kòd jãzëka, na jaczi mô òstac przedolmaczonô.

W Pythonie robi sã to na przëmiôr tak: 

      from xmlrpclib import ServerProxy

      server = ServerProxy("http://open-tran.eu")
      for sugg in server.suggest("save as", "csb"):
          print sugg

Wnetka dóm prosti editork lopków .po, jaczi mdze brëkùjąc nen werk pòbiérôł pòdpòwiescë dolmaczënków apartnëch frazów.
