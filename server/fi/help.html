XML RPC

Voit käyttää tätä palvelua millä tahansa ohjelmalla joka pystyy pyytämään etätoiminnon XML RPC yhteyskäytännön välityksellä. Sinun pitäisi käyttää suggest komentoa, joka ottaa vastaan 2 argumenttia: käännettävä fraasi ja kohde kielen tunnuksen.

Pythonissa voit käyttää sitä seuraavalla tavalla:

      from xmlrpclib import ServerProxy

      server = ServerProxy("http://open-tran.eu/RPC2")
      for sugg in server.suggest("save as", "fi"):
          print sugg

Yritän lähettää mahdollisimman pian yksinkertaisen .po muokkaimen joka tulee kykenemään noutamaan ehdotukset palvelimelta käyttämällä tätä mekanismia.
