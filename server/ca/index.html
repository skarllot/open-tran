Traduir frases individuals

  1. Pots afegir un motor de cerca a qualsevol navegador modern.
  2. No es cap secret que el formulari de cerca va a una URL amb aquest format: http://ca.open-tran.eu/suggest/frase a buscar.

    Ara, introduint una URL del tipus tran frase automàticament s'enviarà una consulta.

XML RPC

Pots utilitzar aquest servei amb qualsevol programa capaç d'executar un procés remot per mitjà del protocol XML RPC. Has d'utilitzar el mètode suggest, que accepta dos arguments: una frase que es vol traduir i el codi de la llengua a la que es vol fer.

Amb Python, pots fer-ho d'aquesta manera: 

      from xmlrpclib import ServerProxy

      lang = "ca" # replace with your language code
      server = ServerProxy("http://open-tran.eu")
      for sugg in server.suggest("save as", lang):
          print sugg
